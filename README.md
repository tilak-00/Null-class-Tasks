# Age and Gender detector internship tasks.
### set up and install.
### Prerequisites
-Python 3.x
-Required Python packages:
- `opencv-python`
  - `dlib`
  - `numpy`
  - `tensorflow`
  - `Pillow`
  - `tkinter`
### Installing Dependencies.
Install the required packages using pip:
```sh
pip install opencv-python dlib numpy tensorflow Pillow
```
